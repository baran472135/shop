# Arasan Shop

Modern Angular tabanlı e-ticaret frontend uygulaması.

## Proje Hakkında

Arasan Shop, Angular 21 ile geliştirilmiş, temiz ve kullanıcı dostu bir alışveriş arayüzüdür. Gerçek zamanlı arama özelliği ile kullanıcıların hızlıca ürünlere ulaşmasını sağlar.

## Özellikler

- Gerçek zamanlı ürün arama (Angular Signals kullanılarak)
- Modern ve kurumsal arayüz tasarımı
- Responsive ürün grid yapısı
- Temiz ve düzenli kod mimarisi
- Standalone Components yaklaşımı

## Kullanılan Teknolojiler

- Angular 21
- TypeScript
- Angular Signals
- Standalone Components
- Bootstrap
